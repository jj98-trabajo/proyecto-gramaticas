# Cuestionario sobre Gramáticas Dependientes de Contexto

Un cuestionario interactivo sobre gramáticas dependientes de contexto, implementado en HTML, CSS y JavaScript.

## Características

- 9 preguntas sobre gramáticas dependientes de contexto
- Interfaz responsive y atractiva
- Retroalimentación inmediata
- Barra de progreso
- Puntuación final

## Tecnologías utilizadas

- HTML5
- CSS3 (con Tailwind CSS)
- JavaScript (ES6)

## Cómo usar

1. Abre `index.html` en tu navegador
2. Responde las preguntas seleccionando una opción
3. Revisa la retroalimentación después de cada respuesta
4. Al final, obtén tu puntuación y un mensaje personalizado
